# Alla - Voice Assistant 🤖

My first Python project! Alla is a personal voice assistant that responds to voice commands.

### ✨ Features:
- Tells current time and date
- Searches Wikipedia & Google
- Opens apps like Notepad  
- Plays YouTube videos
- Responds to "Hello Alla"

### 🚀 How to run:
1. Install dependencies: `pip install -r requirements.txt`
2. Run the assistant: `python Alla_Assistant.py`
3. Say "Hello Alla" to start

### 🛠️ Built With:
- Python
- SpeechRecognition
- pyttsx3
